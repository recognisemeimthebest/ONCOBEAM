# 두경부암 방사선치료 AI 의사결정 지원 EMR — UI 제작용 스펙 요약

> 출처: 팀 기획서 V4.0 (2026-05). 본 문서는 **UI 구현에 필요한 부분만** 추출·정리한 것.
> 시스템은 시간축(치료 전/중/검토) 기준으로 3개 독립 모듈로 구성. 데모 핵심 = 모듈 1 + 모듈 2.

## 0. 한 줄 요약
두경부암 환자의 방사선치료를 ① 치료 시작 전 인과추론(HTE)으로 4개 치료법 비교(모듈 1),
② 치료 중 CBCT 시계열로 종양 부피 변화를 실시간 평가(모듈 2)하여 의사결정을 지원.

## 1. 공통 셸 (Layout)
- **상단 타임라인**: 치료 전(모듈1) / 치료 중(모듈2) / 검토(모듈3) 단계 표시
- **좌측 환자 목록 사이드바**: 환자 ID 키로 선택 (예: C10046)
- **탭 전환**: 선택 환자의 모듈 1 / 모듈 2 화면 전환
- 모듈 독립성: 한 모듈 실패해도 나머지 패널은 정상 표시 (부분 실패 허용)

## 2. 모듈 1 — 치료 시작 전 HTE 사전 예측 (메인)
**입력**
- 환자 EMR: 인구학(sex, birthdate, height, weight), 병기 cancerimaging(T/N/M),
  동반질환(bp 혈압, bs 혈당, sm 흡연, familyhistory), 조직학(classification_cancer),
  진단부위(locationcancer)
- 의사의 치료 계획: 4개 치료법 중 1개 선택 + 치료기법(treatech: conformal/IMRT/기타)
  + 선량(totaldose, radiationcnt)

**4개 치료군 (두경부암 코호트)**
| 코드 | 치료법 | 적용 |
|---|---|---|
| 1 | 방사선 단독 (RT) | T1–T2 조기 |
| 2 | 항암방사선 (CCRT) | T3–T4 또는 N+ 표준 |
| 3 | 수술 + 방사선 | 수술 후 보조 |
| 4 | 수술 + 항암 | 일부 진행기 (⚠️ 표본 적음) |

**출력 (UI에 그릴 것)**
1. 현재 계획의 **3단계 판정 배지**: 효과적 / 불확실 / 비효과적
2. **치료법 쌍별 CATE 비교 바 차트**: 4개 치료법 → 6쌍 비교(C(4,2)=6).
   각 막대에 **Bonferroni 보정 95% CI** (= α/6 ≈ 0.00833 → 99.17% CI) 에러바.
   CI가 0을 포함하면 "불확실(판정 못함)".
3. **SHAP 근거 패널**: 각 피처의 기여도(양/음) 표시
4. **대안 치료 권고**: 자연어 설명

**CATE 개념 (툴팁용)**: "이 환자가 치료 A 대신 B를 받으면 재발률이 얼마나 줄어드는가".
예) 67세 흡연 두경부암: RT→재발 32%, CCRT→재발 24%, CATE = -8%p ("CCRT가 RT보다 8%p 낮춤").

**알고리즘(배경, UI 표기용)**: 성향점수매칭(PSM) → X-learner(메타러너) → Bootstrap CI(n=500)
→ Bonferroni 보정 → 3단계 판정 + SHAP.

## 3. 모듈 2 — 치료 중 실시간 효과 평가
**입력**: 한 환자의 CBCT 시계열(1주차~현재) + EMR 보조 피처
**출력 (UI에 그릴 것)**
1. **자동 BBox 검출 결과 시각화** (YOLOv8, 가장 큰 면적 BBox 1개 추적)
   - 주차별: 검출 슬라이스 수, 평균 면적(px²), 평균 신뢰도
2. **시계열 종양 부피 변화 그래프**: 주차별 부피 + 누적 변화율 + 예측 6주차 값
3. **효과 분류 결과 카드**: 효과적/비효과적(또는 경고) + efficacy_score + 근거 bullets
   + 권고 + 면책 문구

**라벨 정의 (배경)**
- 효과적(1): 6주차까지 부피 ≥30% 감소 AND 재발 없음(relapse=1)
- 비효과적(0): 30% 감소 실패 OR 결국 재발(relapse=2 국소 / 3 원격)
- 제외: followup_days < 180
- 30% 임계값 = RECIST 1.1 PR(직경 30%↓, 부피로는 ~65%↓)보다 민감한 자체 임계값 (조기 신호 포착 목적)

**출력 예시 (C10046, T2N1M0)**
```
[자동 BBox 검출]
1주차: 24/24 슬라이스, 평균 1,200 px², 신뢰도 0.89
4주차: 24/24 슬라이스, 평균   940 px², 신뢰도 0.83
[부피 변화] 1주차 1,200(기준) → 6주차 940(-21.7%) ← 현재
[분석] 주차별 -5.4%/주, 누적 -21.7%, 예측 6주차 -28.5%
[효과 분류] ⚠️ 경고 (efficacy_score = 0.52)
 • 누적 21.7% → 30% 임계값 미달
 • 감소 속도 일정(변동성 낮음) → 긍정 신호
 • T2N1M0 + CCRT 표준 → 평균적 반응 예상
 권고: 5~6주차 추가 관찰 후 재평가, -25% 미만 시 적응형 RT 검토
⚠️ 의사결정 지원용, 최종 판단은 의사.
```

## 5. UI 컴포넌트 ↔ 스펙 매핑 (구현 계획)
```
App (공통 셸)
├─ TopTimeline        ← §1 상단 타임라인 (전/중/검토)
├─ PatientSidebar     ← §1 환자 목록 (ID 키)
├─ ModuleTabs         ← §1 탭 전환
├─ module1/
│  ├─ PatientForm     ← §2 입력 (EMR + 의사 치료계획)
│  ├─ VerdictBadge    ← §2 출력1 3단계 판정
│  ├─ CateBarChart    ← §2 출력2 6쌍 CATE + Bonferroni CI 에러바
│  ├─ ShapPanel       ← §2 출력3 SHAP 기여도
│  └─ RecommendCard   ← §2 출력4 대안 권고
└─ module2/
   ├─ BBoxDetectPanel ← §3 출력1 BBox 검출 시각화
   ├─ TumorVolumeChart← §3 출력2 시계열 부피(+예측) 그래프
   └─ EffectResultCard← §3 출력3 효과 분류 카드
```
